# Instagram DM自動化SaaS 設計メモ（会話まとめ）

## 背景
個人開発で **Instagram支援ツールのSaaS** を開発中。

現状実装済みの機能：

- ユーザー登録
- Instagram連携
- Webhook受信
- DM返信
- コメント返信

現在ある機能：
- **コメントやDMに特定キーワードが含まれていたら固定メッセージを返信**

開発の目的：

> iStepほど重いツールではなく  
> **個人事業主が使いたくなるシンプルなInstagram自動化ツール**

対象ユーザー：

- 個人事業主
- コーチ
- サロン
- スモールビジネス
- 講座販売
- インスタ集客をしている人

---

# プロダクトの基本コンセプト

多機能ツールではなく

**Instagram集客の自動化**

具体的には

```

投稿
↓
コメント
↓
DM
↓
見込み客

```

この導線を自動化するツール。

---

# 必須コア機能（MVP）

以下が最低限必要な機能。

## 1 コメント → DM自動送信

投稿

```

「資料」とコメントしてください

```

↓

ツール

```

DM送信

```

Instagram集客で最も使われる導線。

---

## 2 DMキーワード返信

DM内のキーワードで自動返信。

例

```

資料 → 資料送付
値段 → 料金案内
予約 → 予約URL

```

---

## 3 DMステップ（簡易）

LINEステップの簡易版。

例

```

DM送信
↓
1日後
「資料見ましたか？」
↓
2日後
「相談はこちら」

```

複雑な分岐は不要。

---

## 4 コメントユーザー一覧

投稿ごとの

```

コメントユーザー一覧

```

を表示。

例

```

田中
佐藤
山本

```

運用者はこれを

```

1人ずつDM

```

しているため需要あり。

---

## 5 タグ管理（簡易CRM）

ユーザーにタグ。

例

```

資料請求
相談希望
購入

```

自動タグ例：

```

「資料」とコメント → 資料請求タグ

```
